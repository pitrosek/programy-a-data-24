Svobodný software

V současnosti vystupuje do popředí tzv. svobodný software (free software, nezaměňovat s freeware) šířený většinou pod licencí Open Source (nejčastěji GPL, viz níže). Ten se stává alternativou pro často finančně náročné komerční programy.

Co je to svobodný software?

Svobodný software, známý také jako free software, se vyznačuje tím, že uživatelé mají svobodu spouštět, studovat, sdílet a upravovat software podle svých potřeb.

Svobodný software často používá licenci Open Source, což znamená, že zdrojový kód je otevřený a dostupný pro veřejnost.

Popularita GPL licence

GPL (General Public License) je jednou z nejčastěji používaných licencí pro svobodný software. Tato licence zajistí, že software zůstane svobodný i po jeho úpravách a distribuci.

Podpora pro svobodný software se stává stále rozšířenější, a to zejména díky jeho dostupnosti, bezpečnosti a flexibilitě.
